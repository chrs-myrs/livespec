# E-Commerce Checkout Example

## Why This Exists

Demonstrates LiveSpec's research-to-implementation traceability: how user research findings inform requirements, which guide implementation specs, creating a fully traceable chain from user insights to code.

## What Success Looks Like

- Every requirement traces back to specific research findings
- Requirements justified by user quotes, observations, and measurements
- Complete flow from research artifacts through requirements to implementations
- Realistic scenario based on common e-commerce challenges
